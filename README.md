# Myntra_Clone_Frontend

A Myntra homepage UI clone built using only HTML and CSS. This project recreates the layout and visual structure of Myntra’s landing page including the navbar, banners, product sections, category grid, and a detailed footer. Built for learning and practicing real-world UI cloning with clean structure and styling.


## Project Overview

This clone includes the major Myntra-style sections:
- Header with Myntra logo and navigation categories (Men, Women, Kids, Home, Beauty, etc.)
- Search bar section (image-based UI)
- Main banners and coupon banner
- Product highlight sections like Rising Stars, Luxe Grand Reduction Deals, Medal Worthy Brands, and Global Brands
- Shop By Category image grid
- Footer similar to Myntra’s layout (shopping links, policies, app store badges, social icons, office address)

This is a frontend-only project made to improve HTML structure and CSS positioning skills.

## Technologies Used

- HTML5 (structure)
- CSS3 (styling, layout, positioning)


## Features

- Myntra-style top navbar and menu layout

## How to Run Locally

### Option 1: Open Directly
1. Download or clone this repository.
2. Open `index.html` in your browser.

### Option 2: VS Code Live Server (Recommended)
1. Open the project folder in VS Code.
2. Install the **Live Server** extension.
3. Right-click `index.html` → **Open with Live Server**.
